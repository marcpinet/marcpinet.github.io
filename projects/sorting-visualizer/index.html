Sorting Visualizer

A simple graphical sorter written in Python with Pygame where you can visualize many algorithms and see how they actually work.

You can adjust the speed using ‚Üí or ‚Üê and shuffle the array at the end of the sorting to restart the algorithm by pressing SPACE. Finally, press ESC to exit.

I will add more algorithms when I have time.

How to use

Just launch the main.py, choose your algorithm and the size of the array to sort. When the Pygame window gets opened, press SPACE to start the algorithm.

Prerequisites

  • Python 3.7.0+

Get a copy of the Project. Assuming you have git installed, open your Terminal and enter:

git clone 'https://github.com/marcpinet/sorting-visualizer.git'

To install all needed requirements run the following command in the project directory:

pip install -r requirements.txt
